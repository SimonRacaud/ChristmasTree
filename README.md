# CPool_Tree_2019
Epitech's first-year project.

Display a fir tree with personalized size. 
[Epitech CPool project]

# Marks Epitech
100% Tests / 0 standard error

# Usage
```Bash
$> gcc -o firetree tree.c
$> ./firetree <size>
```
